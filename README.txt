=====================================
  FinPessoal v2.0 – Sistema Financeiro
=====================================

COMO USAR
---------
1. Descompacte este arquivo .zip
2. Abra o arquivo "index.html" no seu navegador
   (Chrome, Edge ou Firefox recomendados)
3. Pronto! O sistema já está funcionando.

Não precisa de internet, servidor ou instalação.


ESTRUTURA DE PASTAS
-------------------
FinPessoal/
├── index.html          ← Abra este arquivo
├── css/
│   └── style.css       ← Estilos visuais
├── js/
│   ├── state.js        ← Estado e constantes
│   ├── utils.js        ← Funções utilitárias
│   ├── charts.js       ← Gráficos SVG
│   ├── modals.js       ← Lógica dos formulários
│   ├── dashboard.js    ← Página Dashboard
│   ├── pagar.js        ← Página A Pagar
│   ├── receber.js      ← Página A Receber
│   ├── dividas.js      ← Página Dívidas
│   ├── cartoes.js      ← Página Cartões
│   ├── relatorios.js   ← Página Relatórios
│   └── app.js          ← Controlador principal
└── README.txt          ← Este arquivo


FUNCIONALIDADES
---------------
✅ Dashboard com KPIs, gráficos e alertas
✅ A Pagar – despesas fixas, parceladas e variáveis
✅ Parcelas automáticas (ex: 10x gera 10 meses)
✅ Despesa fixa recorrente (gera 12 meses)
✅ Marcar como pago, editar e excluir
✅ Alertas de atraso e vencimentos próximos
✅ A Receber – salário, freelance, pessoa, etc.
✅ Receita mensal recorrente (gera 12 meses)
✅ Dívidas parceladas com progresso
✅ Cartões com cor personalizada e barra de uso
✅ Relatórios mensais e histórico
✅ Exportação CSV (mês atual ou histórico completo)
✅ Configurações – nome, meta de economia, alertas
✅ Dados salvos automaticamente no navegador


DADOS
-----
Todos os dados ficam salvos no localStorage do navegador.
Use "Exportar CSV" em Relatórios para fazer backup.

Para transferir para outro computador:
1. Exporte o CSV completo (Relatórios → Histórico completo)
2. Guarde o arquivo CSV como backup


DICA
----
Marque o atalho na barra de favoritos do navegador
para acessar rápido. Funciona offline!
