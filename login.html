Antes de começar

Este é um projeto de portfólio em fase de testes. Preencha seus dados uma única vez — não pedimos senha, é só pra sabermos quem está testando o sistema.

Seus dados financeiros ficam só neste navegador — nunca são enviados a lugar nenhum.

Antes de continuar

Este sistema está em fase de testes e foi liberado apenas para essa finalidade.

Seus dados financeiros ficam salvos somente neste navegador, no aparelho em que você está agora.

Se você trocar de navegador, limpar os dados do site, ou acessar pelo celular, as informações não vão aparecer lá — cada navegador/aparelho guarda os dados de forma independente.
